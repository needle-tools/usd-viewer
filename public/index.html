About USD Viewer

This is a basic USD Viewer created and maintained by Needle, based on Autodesk's USD WASM Fork. It allows you to view Universal Scene Description (USD) files directly in your browser.

Drag and drop USDZ files to load them, or use the samples provided to explore different 3D models.

The actual USD runtime, compiled to WebAssembly, is used here, including all composition features.

We created this viewer while adding OpenUSD support to Needle Cloud.

For more information or to open an issue, please visit the GitHub repo.
